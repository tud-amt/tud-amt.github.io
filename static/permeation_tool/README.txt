# Permeation Tool Website

This repository contains a webpage that serves as a Permeation Tool. It provides explanations, a unit converter, and tools for calculating permeation coefficients. Follow the instructions below to set up and run the webpage.

## Getting Started

1. Clone the repository to your local machine or download the ZIP file and extract it to a folder of your choice.

2. Navigate to the `Permeation_tool` folder, where the main webpage files are located.

3. Open the `Permeation_Tool.html` file in your preferred web browser to access the Permeation Tool.

## File Structure

- `script.js`: JavaScript file containing the logic for the webpage.
- `styles.css`: CSS file containing styles for the webpage.
- `Permeation_Tool.html`: The main HTML file for the Permeation Tool.
- `img/`: A folder containing image files used on the webpage.
- `README.txt`: This file providing instructions and information about the repository.

## Usage

- Once you open the `Permeation_Tool.html` file in your web browser, you will have access to the webpage.
